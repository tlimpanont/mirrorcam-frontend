This page is created to demonstrate that it is possible to change the image dynamically. In order for this to work the previous instance is destroyed and a new one is created with a different image and/or markers, for simplicity we have used four different images but they all have the same graphics with some small modifications.

Please note that the megazoom does not include any API to create thumbnails, instead of thumbnails any HTML tag can be used, for example a HTML button or image. We have chosen thumbnails because in most cases this kind of approach is used.

In the documentation files it is explained in detail how to change the image dynamically on the same page.

This example is included in the download files but we have created a simple example that can be used as reference.
